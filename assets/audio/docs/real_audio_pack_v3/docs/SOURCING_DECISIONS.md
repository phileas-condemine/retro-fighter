# Décisions de sourcing

## Principes retenus

1. Les voix doivent être de vrais enregistrements vocaux, pas des sons synthétiques.
2. Les impacts physiques peuvent venir de packs SFX génériques.
3. Les projectiles doivent être différenciés : métallique/sec pour le shuriken, énergétique/sci-fi pour la boule rose.
4. Les licences CC0 sont prioritaires, mais l’usage local privé permet de garder des options CC-BY / CC-BY-SA dans le manifest.

## Mapping artistique

### Shinobi masculin

- Voix : VoiceBosch `EFFORT SOUNDS (Male)` pour punch/kick/jump/block/hurt/projectile_throw.
- Shuriken : sons de couteau/tranchant Kenney RPG Audio.
- Impact shuriken : metal light Kenney.

### Rose Kunoichi

- Voix principale : `Female RPG Voice Starter Pack`.
- `attack 1/2/3` : punch/kick/projectile_throw.
- `damaged 1/2/3` : hurt.
- `jump 1/2/3` : jump/double_jump.
- `fire/burn/wind/cyclone` : projectile_throw si tu veux un style plus anime/magique.
- Boule rose : Kenney Digital `powerUp`, `phaserUp`, `laser`, `zap`.

## Ce qui reste manuel

Les archives OGA doivent parfois être téléchargées/extraites localement. Freesound exige généralement une connexion. Le script sait préparer les fichiers quand ils sont présents dans `downloads/manual/`.
