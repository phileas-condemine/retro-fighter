# Licences et crédits audio

## Sources principales

### Kenney Sound Pack — CC0
- Impact Sounds, RPG Audio, Digital Audio, Retro Sounds 1/2.
- Licence : Creative Commons CC0.
- Crédit facultatif : `Kenney.nl` ou `www.kenney.nl`.

### Female RPG Voice Starter Pack — CC0
- Auteur : cicifyre.
- Licence : CC0.
- Utilisation recommandée : voix de la `rose_kunoichi`, en particulier `attack`, `damaged`, `jump`, et les mots de sort pour l’attaque à distance.

### VoiceBosch — EFFORT SOUNDS (Male) — CC-BY-SA 4.0
- Auteur/crédit préféré : VoiceBosch.
- Utilisation recommandée : vraie voix masculine du `shinobi_male`.
- Important : si tu redistribues un jeu contenant ces fichiers ou des dérivés, vérifie les obligations CC-BY-SA 4.0.
- Pour un prototype local privé, l’usage est beaucoup moins contraignant, mais garde quand même le crédit.

### 15 vocal male strain/hurt/pain/jump sounds — CC0
- Auteur : qubodup.
- Licence : CC0 depuis 2024-08-30 d’après la page OpenGameArt.
- Utilisation recommandée : fallback masculin strictement CC0.

## Sources optionnelles Freesound

Les sources Freesound listées dans `audio_source_manifest.json` peuvent améliorer la qualité des voix féminines. Freesound exige généralement une connexion pour télécharger. Garde les crédits/licences indiqués sur chaque page.
